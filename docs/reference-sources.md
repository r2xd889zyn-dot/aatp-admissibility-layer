# AATP Reference Sources

## Purpose

Defines how external knowledge is treated within AATP.

---

## Principle

AATP does not assume any external knowledge source is valid by default.

All knowledge is treated as:

→ unverified until admitted

---

## External Knowledge

Includes:

- web content  
- documents  
- academic sources  
- legal texts  
- archives (e.g. libraries, public records)

---

## Admission Rule

No external source is directly used by the system.

All sources must pass through:

- authorship classification  
- admissibility validation  
- anchoring  

---

## Key Distinction

Scale of knowledge ≠ validity

A system can access large volumes of information  
and still operate on invalid input.

---

## Doctrine

Authority is not determined by how much is known.

Authority is determined at the point of admission.

---

## System Flow

External Source  
→ Authorship Layer  
→ Admissibility Layer (AATP)  
→ Anchoring  
→ Context  
→ Reasoning  

---

## Summary

AATP does not integrate knowledge directly.

It governs what knowledge is allowed to become part of the system.
